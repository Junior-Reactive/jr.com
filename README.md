# Junior Reactive Website

## 🚀 Project Overview
Modern full-stack website for Junior Reactive, an AI and IT solutions provider. Built with React, Node.js, Express, and Microsoft SQL Server.

## 📁 Project Structure
- **/backend** - Node.js/Express API with Swagger documentation
- **/frontend** - React application with all pages and components
- **/database** - SQL scripts for database setup

## 🎨 Brand Colors
- Primary Blue: #1c265e
- Secondary Blue: #5269c3
- Accent 1: #90a0da
- Accent 2: #a8ccee
- Light Background: #e4eaf9

## 🔧 Prerequisites
- Node.js v20+
- Microsoft SQL Server (local or cloud)
- npm or yarn
